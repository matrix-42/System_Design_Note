# Database

## Data model

* **Relational**
* **Document**
* **Key value**
* **Wide column**
* Graph

## Storage Engine

* Row oriented \(OLTP\)
  * Hash
  * **LSM-Tree**
  * **B-Tree**
* Column oriented \(OLAP\)

## Examples

* **MongoDB**: Document database using B-Tree engine.
* **Cassandra**: Wide column database using LSM-Tree engine.
